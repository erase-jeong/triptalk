## 👨‍🏫 서비스 개요
한국관광공사의 관광지 오디오 가이드 정보 와 를 활용하여 위API Kakao OpenAPI
치 기반의 관광 정보를 제공하는 대화형 서비스로사용자의 위치를 자동으로 감AI , 
지하여 관련 관광 정보를 오디오로 안내하며음성 인식 기반의 대화형 는 및 , AI UI 
음성 인터랙션 기능을 통해 사용자와 소통한다.
